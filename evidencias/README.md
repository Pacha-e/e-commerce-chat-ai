# Evidencias — Taller 1 Construcción de Software

Capturas de pantalla requeridas por el taller. Cada imagen debe mostrar el nombre del PC, fecha y hora del sistema.

## Capturas requeridas

| Archivo | Descripción | URL/Comando |
|---------|-------------|-------------|
| `01-swagger-ui.png` | Swagger UI con todos los endpoints | http://localhost:8000/docs/ |
| `02-docker-logs.png` | Logs de docker-compose | `docker-compose logs` |
| `03-docker-running.png` | Contenedor activo | Docker Desktop o `docker ps` |
| `04-api-call-products.png` | GET `/api/products/` en Postman | http://localhost:8000/api/products/ |
| `05-api-call-chat.png` | POST `/api/chat/` con respuesta IA | http://localhost:8000/api/chat/ |
| `06-database.png` | BD SQLite con productos | DB Browser / VS Code SQLite |

## Cómo tomar las capturas

1. Iniciar el servidor: `python manage.py runserver` (o `docker-compose up`)
2. Asegurarse de que la barra de tareas muestre la hora del sistema
3. Tomar cada captura con **Win+Shift+S** (Windows Snipping Tool)
4. Guardar en esta carpeta con el nombre exacto de la tabla
